# Fantacode Login & Dashboard (Blazor + .NET Core)

## Features
- JWT-based login
- Protected dashboard with bar chart (Chart.js)
- Designed for distributed systems

## Tech Stack
- Backend: ASP.NET Core Web API
- Frontend: Blazor WebAssembly
- Auth: JWT
- Chart: Chart.js
